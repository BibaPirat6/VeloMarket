# 🚲 VeloMarket

Курсовой проект — веб-сайт по аренде велосипедов с административной панелью.

Проект разработан с использованием нативного PHP без фреймворков и демонстрирует создание полноценного CRUD-приложения с пользовательской частью и системой управления контентом.

## 🌐 Демо

**Сайт:** https://velo.rf.gd/

## ✨ Возможности

### Пользовательская часть

* Просмотр каталога велосипедов
* Просмотр информации о товарах
* Современная адаптивная верстка
* Навигация по разделам сайта

### Административная панель

* Авторизация администратора
* Добавление велосипедов
* Редактирование велосипедов
* Удаление велосипедов
* Управление каталогом

## 🛠️ Используемые технологии

* PHP 8.3
* MySQL 8.0
* HTML5
* CSS3
* JavaScript
* OSPanel

## 📁 Структура проекта

```text
admin/          Панель администратора
assets/         Изображения, стили и скрипты
config/         Конфигурация проекта
database/       SQL-файлы базы данных
includes/       Общие PHP-компоненты
```

> Названия папок могут немного отличаться в зависимости от версии проекта.

## 🚀 Запуск проекта

### 1. Клонировать репозиторий

```bash
git clone https://github.com/BibaPirat6/VeloMarket.git
```

### 2. Разместить проект

Скопировать проект в директорию локального веб-сервера (например, OSPanel).

### 3. Создать базу данных

Создать новую базу данных MySQL.

### 4. Импортировать SQL

Импортировать SQL-файл из проекта в созданную базу данных.

### 5. Настроить подключение

Указать параметры подключения к базе данных в конфигурационном файле.

### 6. Запустить проект

Открыть сайт через локальный сервер.
